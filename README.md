# Evolution of Light

A Roguelike Adventure Game written in Clojure.

## Installation

Clone from https://github.com/kaymanb/eol

## Details

This project is built using [boot](https://github.com/boot-clj/boot).

Run start a new game:

    $ boot start

Run the project's tests:

    $ boot test
